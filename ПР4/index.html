🎯 Концентричні кола

Верстка фігури з task_box_model.png (ініціали ША)
Мінімум двома способами + 1 обов'язковий варіант з одночасним використанням двох псевдоелементів.
Для кожного способу — лише один елемент <div>

— box-shadow

1. Кілька тіней box-shadow (без псевдоелементів)

Центральний елемент — біле коло з текстом. Дві зовнішні тіні створюють жовте та синє кільця. Чудово підходить для простих концентричних фігур.

box-shadow
ША
Розмір: 300 × 300 px (з тінню)
width: 100px; height: 100px;
background: #fff; border-radius: 50%;
box-shadow: 0 0 0 50px #ffff00,
                0 0 0 100px #0066ff;
— radial-gradient

2. Радіальний градієнт (один фон, без псевдоелементів)

Один елемент + background: radial-gradient() з чіткими стопами кольорів. Найкомпактніший спосіб для концентричних кілець.

radial-gradient
ША
Розмір: 300 × 300 px
background: radial-gradient(
  circle at center,
  #ffffff 0% 33%,
  #ffff00 33% 67%,
  #0066ff 67% 100%
);
— ::before + ::after

3. Два псевдоелементи одночасно (::before та ::after)

Обов'язковий варіант за завданням. Зовнішній div — синє коло.
::before — жовте коло всередині.
::after — біле коло з текстом «ША» (центроване через flex).

::before + ::after
Розмір: 300 × 300 px
#variant3 { background: #0066ff; border-radius: 50%; position: relative; }

#variant3::before {
  content: ""; width: 200px; height: 200px;
  background: #ffff00; border-radius: 50%;
  position: absolute; top:50%; left:50%;
  transform: translate(-50%, -50%);
}

#variant3::after {
  content: "ША"; width: 120px; height: 120px;
  background: #fff; border-radius: 50%;
  ... + flex + font styles
}

✅ Всі варіанти використовують тільки один елемент <div> на спосіб.
Кольори підібрані близько до оригіналу task_box_model.png.
Додатково вивчено приклади з PDF «Геометричні фігури на CSS» (кола, border-radius, transform, ::before/::after, box-shadow).
